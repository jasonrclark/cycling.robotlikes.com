# cycling.robotlikes.com

**What:** Visualize your Strava cycling data with interactive charts.

**Run:** Open `index.html` or visit [cycling.robotlikes.com](https://cycling.robotlikes.com)

**Develop:** `ruby serve.rb` (requires Strava API credentials in ENV)

**More:** Issues: https://github.com/jasonrclark/cycling.robotlikes.com/issues
